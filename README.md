# Python豆瓣图书推荐系统简介

## 核心功能
- 登录注册
- 数据爬取
- 数据分析可视化大屏
- 小说信息管理
- 图书借阅管理
- 图书资讯管理
- 用户管理
- 个人中心
- 系统简介

## 适用场景
- 图书馆管理
- 图书资讯平台
- 个人阅读推荐

## 技术特点
- 使用Django框架进行后端开发
- 数据库采用MySQL
- 前端界面采用Vue.js
- 系统具备良好的可扩展性和稳定性
